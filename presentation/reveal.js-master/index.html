What is TypeScript?

Additional layer on JavaScript language


Why it is helpful?

With TypeScript easier to catch bugs during development.

How it works?

TypeScript extends JavaScript by adding types to the language.


Usage example

					
						// Boolean type

						let isCompleted: boolean = false;

						isCompleted = 42;  // Type 42 is not assignable to type boolean
						isCompleted = 'a';  // Type 'a' is not assignable to type boolean
						isCompleted = true;

Assigning types in functions

					
						const sum = (a: number, b: number): number => a + b;

Assigning types in functions

					
						// optional argument

						const createPassword = (name: string, age?: number): string => {
							return `${name}${age}`;
						}

Assigning types in functions

					
						// void

						const greetUser = (name: string): void => {
							alert(`Hello, ${name}`);
						}

Assigning types in functions

					
						// never

						const infiniteLoop = (): never => {
							while (true) {
							}
						}

						const error = (msg: string): never => {
							throw new Error();
						}
